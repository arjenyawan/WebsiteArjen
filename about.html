Tentang Saya

Ini keluarga saya, Mama saya berasal dari Poso,Sulawesi Tengah, Bapa saya berasal dari Biak,Papua, Saya juga Anak pertama dari 3 Bersaudara, ade saya dua-dua nya perempuan

Saya bersekolah di SDN INPRES SARMI,PAPUA

Setelah lulus SD saya melanjutkan SMP di SMPN 1 SARMI,PAPUA

Kemudian setelah lulus SMP saya melanjutkan SMA di SMAN 1 SARMI,PAPUA

Setelah lulus saya sempat ingin masuk IPDN namun ada 1 dan lain hal sehingga saya memilih tes di perguruan tinggi yang pertama UNIVERSITAS CENDRAWASIH, PAPUA dan INSTITUT TEKNOLOGI PLN, JAKARTA dan setelah saya telah menjalani tes, Puji Tuhan saya di terima di kedua Kampus kemudian setelah bicara dengan orangtua saya, saya memilih kampus INSTITUT TEKNOLOGI PLN sebagai kampus saya melanjutkan pendidikan saya
